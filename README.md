# EdgeGuard

## 🔍 소개
**EdgeGuard**는 CPU 사용률을 측정하고 실시간 시각화하는 경량 Linux 기반 시스템 모니터링 툴입니다.  
C++로 작성된 CPU 모니터링 프로그램과 Python 기반 시각화 도구를 통합하여, 시스템 자원 감시에 도움을 줍니다.
